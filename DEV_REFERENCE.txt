================================================================================
  三、架构设计
================================================================================

【整体架构】
  main.py (入口) ──> cli/app.py (菜单路由) ──> brain/agent_brain.py (核心大脑)
        │                                              │
        └── web_panel.py (Web面板)                      ├── api/ (B站API层)
                                                       ├── knowledge/ (知识库)
                                                       ├── persona/ (人格心理)
                                                       ├── services/ (服务模块)
                                                       ├── security/ (安全)
                                                       └── xingye_bot/ (扩展组件)

【核心类: AgentBrain】
  - 位置: brain/agent_brain.py
  - 设计模式: Mixin 组合模式 (13个 Mixin 类横向组合)
  - 主要方法: run() — 启动机器人主循环
  - Mixin 列表:
    1. _BrainInitMixin      (_brain_init.py)      — 初始化
    2. _BrainLoopMixin      (_brain_loop.py)      — 主循环
    3. _BrainVideoMixin     (_brain_video.py)     — 视频理解
    4. _BrainSessionMixin   (_brain_session.py)   — 会话管理
    5. _BrainAIMixin        (_brain_ai.py)        — AI调用
    6. _BrainLearnMixin     (_brain_learn.py)     — 学习归档
    7. _BrainCuriosityMixin (_brain_curiosity.py)  — 好奇心搜索
    8. _BrainAutoMixin      (_brain_auto.py)      — 自动日记/进化
    9. _BrainJournalMixin   (_brain_journal.py)   — 日记系统
   10. _BrainHistoryMixin   (_brain_history.py)   — 视频历史
   11. _BrainUpsMixin       (_brain_ups.py)       — UP主管理
   12. _BrainRuntimeMixin   (_brain_runtime.py)   — 运行时时钟
   13. _BrainInteractMixin  (_brain_interact.py)  — 交互/视觉

【数据流】
  用户操作 -> main.py -> cli/app.py 菜单 -> AgentBrain.run()
    -> _brain_loop.py: B站推荐流获取视频
    -> _brain_video.py: 封面/帧/AI理解 + 评分
    -> _brain_learn.py: 高质量视频 -> 知识提取 -> knowledge/classifier.py 分类归档
    -> knowledge/KnowledgeBase/: 持久化存储
    -> persona/managers.py: 更新人格/心情
    -> persona/psycho.py: 更新心理画像

【配置系统】
  配置文件: Data/config.json (从 config.example.json 复制)
  加载:     core/config.py -> load_config()
  保存:     core/config.py -> save_config() (tmp+replace 原子写入)
  全局变量: core/globals.py — 从 config 动态读取, 通过 from core.globals import * 使用

================================================================================
  十、开发注意事项
================================================================================

1. 【配置修改】修改配置后必须调用 save_config(config) 持久化
2. 【全局变量】修改全局变量后需调用 _reload_all_globals(config) 同步
3. 【JSON写入】所有关键JSON文件必须使用 tmp+replace 原子写入 (防止断电数据损坏)
4. 【Mixin模式】AgentBrain 使用 Mixin 组合, 新增功能需新增 mixin 类并注册到 _mixin_imports.py
5. 【异步调用】所有涉及网络的操作必须在 async 函数中, 使用 asyncio.run() 包装
6. 【API重试】B站API调用应包含重试/退避逻辑
7. 【内容安全】所有AI生成的回复需经过 security/guard.py 审查
8. 【单实例锁】机器人启动时通过 utils/lock.py 确保只有一个实例运行
9. 【多账号】通过环境变量 BILI_ACCOUNT_DATA_DIR 和 BILI_ACCOUNT_NAME 隔离不同账号数据
10.【更新本文件】每次完成任务后必须更新本文件！
